# Welcome

This is the pookiedocs example site. It demonstrates:

- Numeric file prefixes (`01-`, `02-`) that control sidebar order and are stripped from URLs on build
- Plain HTML pages mixed with Markdown pages
- Nested folders containing both `.md` and `.html` files
- Static files served from `staticDir` and referenced in content
