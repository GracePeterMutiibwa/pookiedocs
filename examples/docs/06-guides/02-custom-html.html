Custom HTML Pages

Drop a .html file anywhere in docsDir. pookiedocs extracts the <body> content and wraps it in the site theme - so you get the full sidebar, search, and navigation automatically.

When to use raw HTML

  • Embedding third-party widgets or iframes that need specific markup
  • Tables or layouts too complex for Markdown
  • Migrating existing HTML documentation without rewriting it

Folder nesting

HTML files work inside nested folders just like Markdown files. This page lives at 06-guides/02-custom-html.html and is served at /guides/custom-html - no .html in the URL.
